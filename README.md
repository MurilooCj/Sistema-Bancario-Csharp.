Sistema Bancário em C#

Aplicação de console desenvolvida em C# com integração ao MySQL, com o objetivo de simular operações bancárias essenciais e consolidar conceitos de desenvolvimento back-end.



Funcionalidades

 Cadastro de contas bancárias
 Autenticação de usuários por identificador
 Operações de depósito
 Transferência de valores entre contas
 Armazenamento de saldo em modalidade de "caixinha"


Tecnologias Utilizadas

C#
.NET
MySQL



Características Técnicas

 Implementação de transações no banco de dados (Commit e Rollback), garantindo consistência das operações
 Validação de entradas de dados para maior confiabilidade do sistema
 Separação básica de responsabilidades, visando melhor organização e manutenção do código






Observações

Este projeto possui finalidade educacional e foi desenvolvido com foco na prática de conceitos fundamentais de integração entre aplicação e banco de dados.

Recomenda-se não utilizar credenciais reais diretamente no código-fonte.
