# Sistema-Bancario-Csharp.
 Sistema Bancário em C#  Projeto de console desenvolvido em C# com integração ao MySQL.   Funcionalidades  - Cadastro de conta - Login por ID - Depósito - Transferência entre contas - Sistema de "caixinha"  Tecnologias  - C# - .NET - MySQL  Projeto focado em back-end e manipulacão de banco de dados.
